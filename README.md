# Корпоративний Портал Додатків

Це Streamlit додаток, який служить порталом для різних корпоративних додатків.

## Структура проекту

- `Home.py` - Головний файл додатку
- `pages/` - Сторінки додатків
- `apps_config.json` - Конфігурація доступних додатків
- Підпапки з окремими додатками (BM_v_DOCX, IPNP_v_HTML, MANY_PDF_v_PERSON)

## Особливості

- Автоматичне створення аналітичних досьє з PDF файлів
- Обробка даних з різних джерел (ДМС, Аркан, нерухомість, транспортні засоби)
- **Нова функціональність**: Автоматичний пошук та вставка зображень автомобілів за кольором та маркою/моделлю

## Локальний запуск

1. Клонуйте репозиторій:
   ```
   git clone <repository-url>
   cd streamlit2task
   ```

2. Встановіть залежності:
   ```
   pip install -r requirements.txt
   ```

3. Запустіть додаток:
   ```
   streamlit run Home.py
   ```

## Розгортання на Streamlit.io

1. Створіть публічний репозиторій на GitHub.

2. Завантажте код до репозиторію.

3. Перейдіть на [Streamlit.io](https://streamlit.io/), увійдіть та підключіть репозиторій.

4. Вкажіть головний файл: `Home.py`

5. Додаток буде доступний онлайн.

## Залежності

- Python 3.7+
- Перелічені в `requirements.txt`
- Системні пакети в `packages.txt` (для OCR функціональності)
