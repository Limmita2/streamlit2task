# Конвертация DOCX в PDF

Этот модуль позволяет конвертировать сгенерированные DOCX-документы в PDF с сохранением визуального оформления.

## Установка зависимостей

Для корректной работы конвертации рекомендуется установить системные зависимости для правильного отображения шрифтов:

### Ubuntu/Linux:
```bash
sudo apt install fonts-liberation # Аналог Times New Roman
sudo apt install ttf-mscorefonts-installer # Оригинальные шрифты Microsoft (Arial, Times...)
```

### Windows:
Шрифты по умолчанию уже установлены.

## Использование

После генерации DOCX-документа, он может быть сконвертирован в PDF с помощью функции `convert_docx_to_pdf()`.

## Особенности

- Используется бесплатная библиотека `docxtopdf` для конвертации
- При невозможности использовать docxtopdf, применяется прямое создание PDF с использованием шрифтов, поддерживающих кириллицу
- Сохраняются таблицы, стили и форматирование
- Цветные фоны (например, блакитный цвет для заголовков) сохраняются
- Для корректного отображения шрифтов рекомендуется установка системных шрифтов
- Используются шрифты Liberation Serif или DejaVu Serif, которые поддерживают кириллические символы
